BoviFertil-Pro — Google Sheets bilan ishlaydigan web ilova

Fayllar:
1. index.html — asosiy web ilova
2. style.css — dizayn
3. app.js — xavf hisoblash algoritmi va Google Sheets’ga yuborish kodi
4. google-apps-script.gs — Google Sheets uchun Apps Script kodi

Ishga tushirish:
1. index.html faylini brauzerda oching.
2. Google Sheets’da yangi jadval yarating.
3. Extensions → Apps Script bo‘limini oching.
4. google-apps-script.gs faylidagi kodni Apps Script oynasiga joylang.
5. Deploy → New deployment → Web app ni tanlang.
6. Execute as: Me.
7. Who has access: Anyone.
8. Deploy qiling.
9. Berilgan Web App URL manzilini BoviFertil-Pro ilovasidagi “Google Sheets sozlamasi” qismiga kiriting va saqlang.
10. Formani to‘ldirib “Hisoblash va saqlash” tugmasini bosing.
11. “Oxirgi yozuvni Google Sheets’ga yuborish” yoki “Yuborilmaganlarni Sheets’ga yuborish” tugmasini bosing.

Eslatma:
- Ilova avval ma’lumotni brauzer localStorage xotirasiga saqlaydi.
- Google Sheets’ga yuborish uchun internet va Apps Script URL kerak.
- GitHub Pages’da joylashtirilsa ham ishlaydi.
